Class ServerFileSystemService

IFileSystemService implementation that delegates operations to a CLI server's /api/qcli/fs/* REST endpoints.

Because the IFileSystemService interface is synchronous, this service maintains an in-memory file tree that is returned immediately by read operations. Write operations update the local tree first (for instant feedback) and then fire asynchronous requests to the server for persistence. The initialize() method fetches the initial directory listing from the server.
