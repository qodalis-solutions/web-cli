Interface ICliExecutionHost

Extended execution context interface used internally by the command executor. The concrete implementation (e.g. CliExecutionContext in angular-cli) provides these.

Optional callback that returns the current path to display in the prompt. Any plugin can set this to customize the path segment of the prompt (e.g. a filesystem plugin showing the current working directory). When set, the returned path replaces the default ~ in the prompt. Return null to fall back to the default ~.
