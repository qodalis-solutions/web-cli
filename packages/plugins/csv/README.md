# @qodalis/cli-csv

CSV data manipulation in the terminal

## Installation

```bash
packages add @qodalis/cli-csv
```

## Usage

```bash
csv
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
