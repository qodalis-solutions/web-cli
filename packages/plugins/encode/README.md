# @qodalis/cli-encode

Encoding, decoding, and hashing utilities

## Installation

```bash
packages add @qodalis/cli-encode
```

## Usage

```bash
encode
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
