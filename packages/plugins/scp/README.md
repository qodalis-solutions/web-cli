# @qodalis/cli-scp

SCP-like file transfer between browser and remote servers

## Installation

```bash
packages add @qodalis/cli-scp
```

## Usage

```bash
scp
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
