# @qodalis/cli-chart

ASCII chart and graph rendering

## Installation

```bash
packages add @qodalis/cli-chart
```

## Usage

```bash
chart
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
