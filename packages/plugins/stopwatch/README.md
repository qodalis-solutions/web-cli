# @qodalis/cli-stopwatch

Interactive stopwatch and countdown timer

## Installation

```bash
packages add @qodalis/cli-stopwatch
```

## Usage

```bash
stopwatch
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
