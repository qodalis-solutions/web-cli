# @qodalis/cli-data-explorer

Data Explorer — interactive query console for data sources

## Installation

```bash
packages add @qodalis/cli-data-explorer
```

## Usage

```bash
data-explorer
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
