# @qodalis/cli-wget

Download files from any HTTP/HTTPS URL

## Installation

```bash
packages add @qodalis/cli-wget
```

## Usage

```bash
wget
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
