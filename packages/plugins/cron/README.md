# @qodalis/cli-cron

Schedule commands to run on a repeating interval

## Installation

```bash
packages add @qodalis/cli-cron
```

## Usage

```bash
cron
```

## Development

```bash
npm install
npm run build
```

## Build

Builds two outputs:
- **Module** (CJS + ESM): `dist/public-api.js` and `dist/public-api.mjs`
- **IIFE bundle**: `dist/umd/index.js` (for browser runtime loading)
