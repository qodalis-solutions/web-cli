Architecture 
Frontend CLI  <-- REST / WebSocket -->  Backend Server
(Browser)                                 (Node.js / Python / .NET)
  |                                          |
  |-- POST /api/v1/cli/execute ----------->  |-- Execute command
  |-- GET  /api/v1/cli/commands -----------> |-- List commands
  |-- WS   /ws/v1/cli/shell ---------------> |-- PTY shell session
  |-- WS   /ws/v1/cli/events --------------> |-- Event streaming
  |-- GET  /api/cli/fs/* ------------------> |-- Filesystem access
