Architecture 
Frontend CLI  <-- REST / WebSocket -->  Backend Server
(Browser)                                 (Node.js / Python / .NET)
  |                                          |
  |-- POST /api/v1/qcli/execute ----------->  |-- Execute command
  |-- GET  /api/v1/qcli/commands -----------> |-- List commands
  |-- WS   /ws/v1/qcli/shell ---------------> |-- PTY shell session
  |-- WS   /ws/v1/qcli/events --------------> |-- Event streaming
  |-- GET  /api/qcli/fs/* ------------------> |-- Filesystem access
