Getting Started

Qodalis CLI is a multi-framework terminal component for the web. It provides a fully interactive xterm.js-based terminal with a plugin architecture, built-in commands, and native wrappers for Angular, React, and Vue.

Installation

Choose your framework and install the corresponding package:

npm 
npm install @qodalis/angular-cli
npm 
npm install @qodalis/react-cli
npm 
npm install @qodalis/vue-cli

Quick Start: Angular

Import CliModule in your Angular module:

app.module.ts 
import { CliModule } from '@qodalis/angular-cli';

@NgModule({
    imports: [CliModule],
})
export class AppModule {}

Add the stylesheet to your angular.json styles array:

angular.json 
"styles": [
    "node_modules/@qodalis/angular-cli/src/assets/styles.sass"
]

Add the terminal to your template:

app.component.html 
<!-- Full terminal -->
<cli></cli>

<!-- Collapsible panel -->
<cli-panel></cli-panel>

Quick Start: React

Import and use the Cli or CliPanel component:

App.tsx 
import { Cli, CliPanel, CliConfigProvider } from '@qodalis/react-cli';

function App() {
    return (
        <CliConfigProvider modules={[]}>
            {/* Full terminal */}
            <Cli />

            {/* Or collapsible panel */}
            <CliPanel />
        </CliConfigProvider>
    );
}

Quick Start: Vue

Import and use the Cli or CliPanel component:

App.vue 
<script setup>
import { Cli, CliPanel, CliConfigProvider } from '@qodalis/vue-cli';
</script>

<template>
    <CliConfigProvider :modules="[]">
        <!-- Full terminal -->
        <Cli />

        <!-- Or collapsible panel -->
        <CliPanel />
    </CliConfigProvider>
</template>

Adding Plugins

Install a plugin package and add it to your modules array:

npm 
npm install @qodalis/cli-guid
Import and use 
import { guidModule } from '@qodalis/cli-guid';

// Angular: <cli [modules]="[guidModule]"></cli>
// React:   <Cli modules={[guidModule]} />
// Vue:     <Cli :modules="[guidModule]" />

You can also install plugins at runtime without rebuilding:

Terminal 
pkg add @qodalis/cli-guid
